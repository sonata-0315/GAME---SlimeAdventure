# Tile Sprites

This folder will contain simple colored square sprites for platforms.

In Unity, create these sprites using the Sprite Editor:
1. Create → 2D → Sprites → Square
2. Set Pixels Per Unit to 100
3. Create color variations for zones:
   - White (default platform)
   - Red (Zone A - Movement testing)
   - Blue (Zone C - Jump testing)
   - Green (Zone E - Wall testing)
   - Yellow (Zone D - Gap testing)
   - Orange (Zone G - Precision testing)
   - Purple (Zone H - Chain testing)

For now, we'll use Unity's built-in white sprite and color it with SpriteRenderer.
